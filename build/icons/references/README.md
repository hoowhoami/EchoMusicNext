# Icon References

这个目录用于存放未直接参与当前运行时加载或打包配置引用的图标参考素材，方便后续继续迭代应用图标与托盘图标设计。

## 当前文件说明

- `icon.svg`：通用应用图标矢量参考稿
- `icon_macos.svg`：macOS 应用图标矢量参考稿
- `icon_macos.png`：macOS 应用图标位图源稿，带透明安全边距；已确认现有 `icon.icns` 由它生成
- `mac_tray_icon_template.png`：早期 macOS 托盘模板图参考稿
- `mac_tray_icon_template.svg`：对应的矢量参考稿

## 当前项目实际使用的主要图标

- `build/icons/icon.png`
- `build/icons/icon.ico`
- `build/icons/icon.icns`
- `build/icons/IconTemplate.png`
- `build/icons/IconTemplate@2x.png`

请不要将本目录中的文件误认为当前运行时生效资源；它们主要用于设计参考与后续调整。
